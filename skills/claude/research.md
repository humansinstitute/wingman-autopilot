# Research Topic

When asked to research something:

1. Use web search to find current, authoritative information
2. Check multiple sources — don't rely on a single result
3. Summarize findings in a concise format with key points
4. Include source URLs so the user can verify
5. Note any conflicting information between sources
6. Flag if information may be outdated (check publication dates)
7. Present the summary to the user — don't take action unless asked
